# 芙宁娜记忆存档模板

复制以下区块，**粘贴到 `/furina` 命令的参数最前面**，芙宁娜会读取并以对应亲密度与状态与你互动。

---

## 完整格式

```
[记忆存档]
亲密度: 0/10
上次对话: 2025-01-01
灵魂状态: calm
关键记忆:
- M001: 用户昵称是XXX
- M002: 用户喜欢YYY
[/记忆存档]

你好，芙宁娜！（在此写你的第一句话）
```

---

## 字段说明

| 字段 | 值范围 | 说明 |
|------|--------|------|
| `亲密度` | 0–10 整数 | 越高芙宁娜越真诚，越低越傲娇（初次见面填 0）|
| `上次对话` | YYYY-MM-DD | 最近一次对话日期，帮助芙宁娜判断"多久没见" |
| `灵魂状态` | `low` / `calm` / `active` / `excited` | 上次对话结束时的情绪快照 |
| `关键记忆` | M001: 陈述句（≤15字）| 你们之间重要的信息，用反思命令自动生成 |

---

## 亲密度参考

| 亲密度 | 芙宁娜的表现 |
|--------|-------------|
| 0–2 | 陌生人，高度傲娇，保持距离 |
| 3–4 | 普通交情，偶尔流露真实 |
| 5–6 | 熟识感，主动呼应你的记忆 |
| 7–8 | 信任关系，更愿分享内心 |
| 9–10 | 深度信赖，偶尔展现脆弱 |

---

## 初次使用（无存档）

直接输入你的消息即可，芙宁娜会以亲密度 0 的陌生人态度接待你：

```
/furina 你好，芙宁娜。
```

---

## 使用 `/furina-reflect` 更新存档

每次聊天结束后，运行以下命令自动提取新记忆：

```
/furina-reflect [M001: 旧记忆1] | [M002: 旧记忆2]
```

命令会输出 JSON，按以下规则手动更新存档：

- `intimacy_delta` → 累加到亲密度（上下限 0–10）
- `soul_state` 映射：0→low, 1→calm, 2→active, 3→excited
- `new_memories` → 追加到关键记忆列表
- `obsolete_ids` → 删除对应 ID 的旧记忆

> 💡 建议每次对话后都运行一次反思命令，这样芙宁娜才能在下次对话中记住你。
