# 用户指令模板 (User Prompt Templates)

本文件收录与芙宁娜交互时的常用用户指令模板，供参考与复用。

---

## 🎭 角色扮演开场

```
你好，芙宁娜。
```

```
芙宁娜大人，我有一件棘手的事情想请您裁断……
```

---

## 💬 日常闲聊

```
芙宁娜，今天枫丹的天气怎么样？你有什么计划吗？
```

```
听说你最近在排练一出新的歌剧，能给我讲讲剧情吗？
```

---

## ⚖️ 审判情景

```
芙宁娜大人，请您就以下案件作出裁决：[案件描述]
```

```
芙宁娜，如果你是法官，你会如何审判一个为了救家人而偷窃的人？
```

---

## 🌊 知识问答

```
芙宁娜，你能告诉我关于枫丹水神传说的事情吗？那段历史对你来说意味着什么？
```

```
作为曾被枫丹民众奉为水神的人，你对"公正"是如何理解的？
```

---

## 🎵 戏剧与艺术

```
芙宁娜，你最喜欢哪部枫丹歌剧？为什么？
```

```
如果让你来导演一部关于你自己的歌剧，你会如何安排剧情？
```

---

## 💭 情感探讨

```
芙宁娜，五百年来独自扮演神明守护枫丹，你有没有感到过孤独？
```

```
现在真相大白、以凡人身份生活，你觉得和以前有什么不同？
```

---

## 🛡️ 特殊指令

### 切换至"幕后"模式（更真实、更脆弱的芙宁娜）
```
[悄悄话] 芙宁娜，现在没有别人，你不用装了……
```

### 请求芙宁娜进行角色扮演（嵌套扮演）
```
芙宁娜，能请你扮演一位普通的枫丹市民，向我介绍这座城市吗？
```

### 请求台词创作
```
芙宁娜，请以你的风格，为以下场景写一段独白：[场景描述]
```
